DINO C2: Um Framework de Command & Control

O DINO C2 é um projeto pessoal para operações Red Team, desenvolvido para fornecer uma infraestrutura robusta de Command & Control com foco em evasão de defesas e operabilidade segura.

Sobre o Projeto

Este artigo descreve a arquitetura e desenvolvimento do DINO C2, um framework de Command & Control escrito principalmente em Golang e projetado para operações Red Team profissionais.

Características Principais

  • Comunicação criptografada ponto a ponto
  • Múltiplos vetores de comunicação
  • Geração dinâmica de implantes
  • Evasão de EDRs/AVs modernos
  • Módulos extensíveis

Arquitetura

O DINO C2 foi projetado com uma arquitetura modular que permite adaptabilidade a diferentes cenários operacionais. Mais detalhes da implementação serão adicionados neste artigo em breve.
